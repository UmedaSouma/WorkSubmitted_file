-------------------------------------------------------------------
◆このゲームについて
-------------------------------------------------------------------

敵の力を使ってステージを攻略しよう

右に進んでく横スクロールアクション。
敵を倒して、ピンクのかけらを取得すると敵が仲間になる。
一番右にいるボスを倒してクリア

-------------------------------------------------------------------
◆起動方法
-------------------------------------------------------------------

exeからゲームを起動


-------------------------------------------------------------------
◆基本操作
-------------------------------------------------------------------

・キーボード

[A/D]		// 左右移動
[SPACE]		// ジャンプ
[E]			// スキル
[Enter]		// 剣攻撃

・コントローラー

[左スティックを左右に倒す]		// 左右移動
[A]						// ジャンプ
[Y]						// スキル
[RT]						// 剣攻撃
